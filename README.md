This project is a small JavaScript learning repository built inside the `notes` folder. It currently focuses on core beginner topics such as variables, scope, hoisting, data types, truthy and falsy values, type coercion, and common JavaScript quirks.

## Project Structure

```text
notes/
  Lesson1/
  Lesson2/
```

## What This Project Covers

### Lesson 1: Variables and Scope

Topics covered in `notes/Lesson1`:

- Variables in JavaScript
- `var`, `let`, and `const`
- Hoisting
- Temporal Dead Zone (TDZ)
- Scope
- Reassignment, reinitialization, and redeclaration

Files:

- [Variables](notes/Lesson1/variable.md)
- [Hoisting](notes/Lesson1/hosting.md)
- [Temporal Dead Zone](notes/Lesson1/tdz.md)
- [Scopes](notes/Lesson1/scopes.md)
- [Reassignment and Redeclaration](notes/Lesson1/reassign-redeclare.md)

### Lesson 2: Types and JavaScript Behavior

Topics covered in `notes/Lesson2`:

- JavaScript data types
- Dynamic typing
- `typeof` operator
- Truthy and falsy values
- Type coercion
- JavaScript quirks

Files:

- [Data Types](notes/Lesson2/datatypes.md)
- [Dynamic Typing](notes/Lesson2/dynamic-typing.md)
- [Typeof Operator](notes/Lesson2/typeof-operator.md)
- [Truthy and Falsy Values](notes/Lesson2/truthy-and-falsy-values.md)
- [Type Coercion](notes/Lesson2/type-coercion.md)
- [JavaScript Quirks](notes/Lesson2/quirks-in-js.md)

## Key Ideas From The Notes

- Prefer `const` when the value should stay the same.
- Use `let` when reassignment is needed.
- Avoid `var` because it can create confusing behavior with scope and redeclaration.
- Understand hoisting and TDZ before using variables early in code.
- Learn how JavaScript handles primitive and reference types.
- Practice truthy/falsy checks and type coercion because they affect real program behavior.

## Purpose

The purpose of this project is to organize JavaScript fundamentals into simple lesson-based notes for study and revision.
