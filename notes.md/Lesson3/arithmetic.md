>  <h1><center>Arithmetic Operators

`These are used to perform standard mathematical calculations.`

```java
+    //addition
-    //substraction
*    //multiplication
/    //division
%    //modulus
**   //exponentiation
```
`Addition (+) is used for numeric addition.`

```js       
let sum = 5 + 3; // sum is 8
```

`Also know as concatenation (+) is used for joining strings.`

```js       
let greeting = "Hello" + " " + "World"; // greeting is "Hello World"
```

`Subtraction (-) is used for numeric subtraction.`

```js       
let difference = 5 - 3; // difference is 2
```
`Multiplication (*) is used for numeric multiplication.`

```js       
let product = 5 * 3; // product is 15
````
`Division (/) is used for numeric division.`

```js       
let quotient = 5 / 3; // quotient is approximately 1.6667
```
`modulus (%) is used for finding the remainder after division.`

```js       
let remainder = 5 % 3; // remainder is 2
```
`Exponentiation (**) is used for raising a number to the power of another.`$$5^{3} = 125$$

```js   
let power = 5 ** 3; // power is 125
```
