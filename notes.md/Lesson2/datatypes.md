> <h1> <center>DATATYPES


`In JavaScript, every value has a type. These types define what kind of data is being stored — a number, text, boolean, object, etc. There are two categories:`


**Primitive types – stored directly.** <br>
**Reference types – stored as memory references.**


> <h2>Primitive Data Types <br>


`String → Text`
   ```js
   "hello" , 'Rutaab'
   ```

`Number → Any numeric value`
   ```js
   3 , -99 , 3.14
   ```

`Boolean → True or false`
   ```js
   true , false
   ```

`Undefined → Variable declared but not assigned`
   ```js
   let x;   //x is undefined
   ```

`Null → Intentional empty value`
   ```js
   let x = null;
   ```

`Symbol → Unique identifier(rarely used)`
   ```js
   Symbol()
   ```

`BigInt → Very large integers` 
   ```js        
   123456789012345678901234567890n    
   ```

> <h2>Reference Data Types

`Object`
   ```js
   { name: "Rutaab", age: 21 }
   ```

`Array`
   ```js 
   [10, 20, 30]
   ```

`Function`
   ```js 
   function greet()
   {
      code
   }
```
