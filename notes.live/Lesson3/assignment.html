Assignment Operators

These assign values to variables. The most common is =, but there are "compound" versions that perform an operation and assign the result at the same time.

=    //assignment operator
+=   // Addition assignment
-=   // Subtraction assignment
*=   // Multiplication assignment
/=   // Division assignment
%=   // Modulus assignment

How compound assignment operators really work:

a = 5;          // a is assigned the value 5
a += 3;         // compound assignment applied here  

//it means add 3 to the current value of a that is 5 and then assign the result back to a

a = a + 3 = 8;  // a is now assigned the value 8

// same happens with other compound assignment operators

Assignment operator (=) is used to assign a value to a variable.

let a = 5; // a is assigned the value 5

Addition assignment (+=) adds a value to the variable and assigns the result back to the variable.

let a = 5; // a is assigned the value 5
a += 3;    // a is now assigned the value 8

Subtraction assignment (-=) subtracts a value from the variable and assigns the result back to the variable.

let a = 5; // a is assigned the value 5
a -= 3;    // a is now assigned the value 2

Multiplication assignment (*=) multiplies the variable by a value and assigns the result back to the variable.

let a = 5; // a is assigned the value 5
a *= 3;    // a is now assigned the value 15

Division assignment (/=) divides the variable by a value and assigns the result back to the variable.

let a = 5; // a is assigned the value 5
a /= 3;    // a is now assigned the value approximately 1.6667

Modulus assignment (%=) takes the modulus of the variable by a value and assigns the result back to the variable.

let a = 5; // a is assigned the value 5
a %= 3;    // a is now assigned the value 2
