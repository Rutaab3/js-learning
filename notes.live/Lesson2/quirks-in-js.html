QUIRKS IS JAVASCRIPT

Quirks are the unusual or unexpected output or behaviour of javascript. It part of the langauge design.

Here are 10 JavaScript quirks:

NaN === NaN            `returns`     false
[] + []                `returns`     ""
{} + []                `returns`     0
true + true            `returns`     2
null == undefined      `returns`     true
0.1 + 0.2 !== 0.3      `returns`     true
typeof NaN             `returns`     "number"
[] == ![]              `returns`     true
3 > 2 > 1              `returns`     false
1(!+[]+[]+![]).length  `returns`     9

These are some unusual output of some equations you can practice more yourself.
