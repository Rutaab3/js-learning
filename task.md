### Level: Basic (Variables & Strings)
1.  **The Name Corrector:** Create a variable `userName` with any name in all lowercase (e.g., `"ali"`). Create a second variable that stores the name with only the first letter capitalized.
2.  **The Temperature Converter:** Create a variable `celsius`. Write a formula to convert it to Fahrenheit and log the result. (Hint: $F = C \times \frac{9}{5} + 32$)

### Level: Intermediate (If/Else & Booleans)
3.  **The Karachi Traffic Signal:** Create a variable `signalColor`. 
    * If "Red", log "Stop". 
    * If "Yellow", log "Slow down". 
    * If "Green", log "Go". 
    * If anything else, log "Signal Malfunction".
4.  **The Exam Grader:** Create a variable `score` (0-100). 
    * 90+ is "A+". 
    * 80-89 is "A". 
    * 70-79 is "B". 
    * Below 70 is "Fail". (Use `if/else if` chains).
5.  **The ATM Logic:** Create two variables: `accountBalance` and `withdrawalAmount`. 
    * If `withdrawalAmount` is greater than `accountBalance`, log "Insufficient Funds".
    * Otherwise, log "Transaction Successful" and log the new balance.

### Level: Elite (Switch Cases & Logic Nesting)
6.  **The Mobile Plan Switch:** Create a `planCode` variable ("PRE", "POST", "DATA"). 
    * Use a `switch` to log different prices. 
    * Group "PRE" and "POST" together to log "Voice Plan Active".
7.  **The Weekend Warrior:** Create a variable `day`. Use a `switch` statement to log "Weekday" for Monday-Friday and "Weekend" for Saturday/Sunday using **fall-through** logic.
8.  **The Cinema Ticket:** Create `age` and `isStudent` (boolean). 
    * If `age < 12`, price is 500 PKR.
    * If `age >= 12` AND `isStudent` is true, price is 800 PKR.
    * Else, price is 1200 PKR.

### Level: The "Invincible" Brain Teaser
9.  **The Smart Home:** Create a variable `isNight` and `motionDetected`. 
    * Log "Lights ON" only if it is night **AND** motion is detected. 
    * Log "Lights OFF" in any other case.
10. **The Login Validator:** Create `db_user = "admin"` and `db_pass = "12345"`. Create `input_user` and `input_pass`. Use an `if` statement to check if both match. 
    * If yes, log "Welcome Admin". 
    * If user matches but password doesn't, log "Wrong Password". 
    * Otherwise, log "User Not Found".
