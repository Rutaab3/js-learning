Learning Journey

Build confidence from variables to operator logic.

The repository moves from foundations to behavior, then into expressions and operators so each lesson builds on the last one naturally.

01

Variables and scope

Start with `var`, `let`, `const`, hoisting, TDZ, and how scope affects access.

02

Types and coercion

Understand dynamic typing, `typeof`, truthy and falsy values, and JavaScript quirks.

03

Operators in action

Move into arithmetic, comparison, logical, unary, ternary, and assignment operators.

04

Control flow patterns

Finish with `if`, `else if`, `switch`, logical patterns, and the return early style.
